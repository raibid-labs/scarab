## Theme Selection Guide

### By Use Case

**For Coding:**
- Dracula (dark)
- Monokai (dark)
- One Dark (dark)
- One Light (light)

**For Long Sessions:**
- Solarized Dark/Light (scientifically balanced)
- Catppuccin Mocha/Latte (soft pastels)
- Nord (calm, minimal)

**For Aesthetics:**
- Tokyo Night (modern, clean)
- Synthwave (retro, fun)
- Gruvbox (vintage)

**For Daylight:**
- Solarized Light
- One Light
- Catppuccin Latte

**For Night:**
- Dracula (default)
- Tokyo Night
- Nord

### By Color Preference

**Purple Lovers:** Dracula, One Dark
**Blue Lovers:** Nord, Tokyo Night
**Warm Colors:** Monokai, Gruvbox
**Pastels:** Catppuccin
**Neon/Vibrant:** Synthwave
**Balanced:** Solarized

### Quick Switch Commands

```bash
# Via command palette (Ctrl+Shift+P)
Theme: Select

# Or directly in config
[colors]
theme = "your-choice-here"
```
