# Interactive Tutorial

Scarab includes an interactive tutorial that guides you through its features.

## Quick Links

For tutorial information, see:
- [Getting Started Tutorial](../../../tutorials/01-getting-started.md) - Tutorial documentation

## Launching the Tutorial

### On First Run

The tutorial launches automatically when you first start Scarab. You can:
- Press **Space** or **Enter** to progress through steps
- Press **ESC** to skip the tutorial

### Replay Anytime

To replay the tutorial after your first run:

```bash
scarab-client --tutorial
```

## Tutorial Contents

The interactive tutorial covers:

1. **Welcome** - Introduction to Scarab's architecture
2. **Navigation** - Basic terminal commands and navigation
3. **Scrollback** - Using mouse wheel and keyboard for scrolling
4. **Link Hints** - Keyboard-driven URL opening with Ctrl+Shift+O
5. **Command Palette** - Quick command access with Ctrl+Shift+P
6. **Plugins** - Overview of the plugin system
7. **Configuration** - Customization basics and config location
8. **Completion** - Next steps and resources

## Tutorial Features

- **Step-by-step guidance** with visual highlights
- **Interactive demonstrations** of features
- **Hands-on practice** for key commands
- **Skip or replay** at any time
