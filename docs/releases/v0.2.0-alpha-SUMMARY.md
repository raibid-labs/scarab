## Quick Commands

### Test Everything
```bash
just test-all              # Comprehensive test suite
just ci                    # Format, clippy, test
```

### Run Specific Tests
```bash
just test                  # Unit tests
just nav-smoke             # Navigation smoke tests
just golden                # Visual regression tests
just testlib               # ratatui-testlib tests
just headless              # Headless harness tests
just e2e                   # E2E integration tests
```

### Build Documentation
```bash
just docs-build            # Build mdBook
just docs-serve            # Serve locally with live reload
```

### Release Preparation
```bash
cargo check --workspace    # Verify compilation
cargo clippy --workspace   # Check for warnings
cargo audit                # Security audit
cargo build --release      # Release build
```
