## Changelog Draft

### Added
- **ECS-native Navigation System**
  - `NavFocus`, `NavHint`, `NavMode` components for state management
  - Per-pane navigation state isolation
  - Vimium-style hint-based navigation (f-key activation)
  - Navigation input router with configurable keybindings
  - Navigation metrics for performance tracking

- **Plugin Navigation API**
  - Security guardrails for plugin-sourced focusables
  - Rate limiting for plugin actions (10/sec default)
  - `register_focusable` and `unregister_focusable` API endpoints
  - Navigation mode control for plugins

- **Testing Infrastructure**
  - ratatui-testlib integration for smoke tests
  - Comprehensive TESTING.md guide
  - Test harness for widget verification
  - Component isolation test patterns

- **Documentation Portal**
  - mdBook-based documentation site (docs/book/)
  - Navigation user guide with quickstart examples
  - Navigation developer guide with ECS patterns
  - Contributing guide for documentation (CONTRIBUTING-DOCS.md)

- **Telemetry & Debugging**
  - Telemetry HUD plugin for real-time performance monitoring
  - FPS, frame time, and navigation metrics display
  - Hot-reloadable telemetry overlay
  - Configurable telemetry settings

### Changed
- **Navigation Architecture**
  - Moved from plugin-based to ECS-native implementation
  - Navigation state now stored in Bevy components instead of shared state
  - Improved performance and maintainability with ECS patterns

- **Documentation Consolidation**
  - Organized documentation into docs/README.md index
  - Restructured guides into logical directories (navigation/, guides/, etc.)
  - Created single source of truth for navigation documentation

### Fixed
- **Navigation State Isolation**
  - Fixed navigation state leaking between panes
  - Each pane now maintains independent navigation state via `PaneId`
  - Proper cleanup of navigation state when panes are destroyed

- **Plugin Security**
  - Added validation for plugin-sourced focusables
  - Rate limiting prevents plugin abuse of navigation API
  - Bounded storage for plugin-registered focusables

### Known Issues
- Daemon IPC handler missing match arms for new navigation control messages (will be fixed before tag creation)
- Various unused code warnings (non-blocking, tracked for future cleanup)
