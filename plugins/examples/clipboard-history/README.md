# Clipboard History Plugin

Tracks clipboard history and provides quick access to previously copied items. Frontend plugin for fast UI interaction.

## Features

- Stores last 50 clipboard items
- Prevents duplicates
- Persistent across sessions
- Quick search and paste

## Installation

```bash
just plugin-build clipboard-history
```

## Usage

The plugin tracks clipboard automatically. Future versions will add keyboard shortcuts to show history.
