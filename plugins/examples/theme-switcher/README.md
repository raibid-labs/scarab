# Theme Switcher Plugin

Quick theme switching with live preview. Change your terminal appearance on the fly.

## Features

- 8 built-in themes
- Live preview
- Persistent theme selection
- Keyboard shortcuts (planned)

## Installation

```bash
just plugin-build theme-switcher
```

## Available Themes

- Dracula
- Monokai
- Solarized (Dark/Light)
- Gruvbox
- Nord
- Tokyo Night
- Catppuccin
